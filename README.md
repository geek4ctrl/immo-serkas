# Immo Serkas - Angular Website

A modern Angular 21 website for Immo Serkas Real Estate, featuring Home, About, Services, and Contact pages.

## Pages

- **Home** (`/home`) - Hero, Welcome, Objectives, Features, and Services sections
- **About** (`/about`) - Company information and founder profile
- **Services** (`/services`) - Service offerings
- **Contact** (`/contact`) - Contact form, info, and map

## Run the Application

```bash
cd immo-serkas-app
npm start
```

Then open [http://localhost:4200](http://localhost:4200) in your browser.

## Build for Production

```bash
cd immo-serkas-app
npm run build
```

The built files will be in `dist/immo-serkas-app/`.
